# [Instalação](#índice)
1. Clone este repositório: `https://github.com/anabeatrizzz/mobile2-app-quatro.git`

2. Entre na pasta do repositório pelo terminal

3. Inclua as dependencias: `npm install`

4. Execute o projeto: `npm start`
